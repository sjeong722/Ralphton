# ThinkGym Submission Bundle

This bundle contains only files required to run the ThinkGym `/session` demo with mock backend.

## Run
1. npm install
2. npm run dev
3. Open http://localhost:3000/session

## Notes
- Backend engine: `backend/run.py` (mock-first, JSON contract)
- API routes: `app/api/*`
- Session UI: `app/session/*`
