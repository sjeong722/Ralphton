# AGENTS.md

## 1. 수정 범위 (Scope)
- 수정 허용: .claude/ , .agents/
- 수정 금지: (현재 폴더 구조상 해당 경로 없음)
- 기본 원칙: 위에 명시되지 않은 경로는 변경 전 반드시 승인 요청

## 2. 작업 절차 (Workflow)
- 작업 후 확인: 내용 정확성 검토, 누락 항목 확인, 형식 점검
- 승인 규칙: 외부 전송 시 승인 필수, 원본 데이터 수정 절대 금지

## 3. 보고 형식 (Report Format)
- 변경 파일 목록
- 변경 내용 요약 (3줄 이내)
- 검증 결과 (Pass/Fail)
- 주의사항 (사람의 확인이 필요한 부분)

## 4. MCP 보안 규칙
- 기본 원칙: 모든 외부 도구 연결은 Read-only로 시작
- 쓰기 권한: 되돌릴 수 없는 작업은 사용자 명시 승인(on-request) 필수

## 5. 롤백 및 사고 대응
- 사고 감지/중지 요청 시: 즉시 작업 중지 → 영향 파악 → 마지막 안정 상태로 롤백 → 보고
- Git 사용 시: 필요하면 사용자 승인 후 롤백 명령 수행
