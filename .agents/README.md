# .agents 운영 가이드

이 디렉터리는 에이전트 실행 규약을 분리 관리합니다.

## 디렉터리 구조
- `skills/`: 재사용 가능한 작업 절차 정의
- `hooks/`: 이벤트 기반 자동 점검/가드레일
- `commands/`: 사람이 호출하는 표준 명령 모음
- `thinkgym-mini/`: 프로젝트별 구현 자산

## 연결 원칙
- 사람용 기준은 `skills`에 둡니다.
- 자동 실행은 `hooks`에서만 담당합니다.
- 수동 실행 진입점은 `commands`에서만 제공합니다.
- 동일 로직을 중복 구현하지 않고, 스크립트 1개를 공유 호출합니다.
