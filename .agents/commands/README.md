# Commands 초안

`commands`는 사용자가 명시적으로 호출하는 표준 작업 진입점입니다.

## 목표
- 반복 작업을 동일한 입력/출력 규격으로 실행
- 스킬 워크플로를 빠르게 호출
- 결과 보고 형식을 강제

## 구성
- `commands.example.yaml`: 명령-스크립트 매핑 초안
- `scripts/`: 명령 실행 스크립트
- `.claude/commands/*.md`: 실제 slash command 진입점

## 운영 규칙
- 각 명령은 담당 스킬을 1개 이상 명시
- 표준 출력은 `요약/파일목록/검증/주의사항` 순서 유지
- 실패 시 즉시 종료하고 복구/재시도 힌트 제공
